# My_WebServer
一个使用C++编写的Linux下轻量级Web服务器  
1.采用事件驱动模型，使用主从reactor模型，主时间循环负责监听和和分发给sub_reactor，使得有效处理多个并发连接  
2.采用非阻塞式的IO  
3.使用了缓冲区的操作，用于管理数据的读写  
4.添加了线程池，有效的管理和利用多线程资源  
5.使用状态机解析Http报文，支持多种模式的解析    
6.采用现代C++实现，尽量的使用了C++新特性
